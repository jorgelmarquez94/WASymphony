Symphony String Methods

In this tutorial you will learn how to create strings using the str.

String Methods with Symphony

Great work! Now that we know how to store strings, let's see how we can change them using string methods. String methods let you perform specific tasks for strings. We'll focus on four string methods:

  • length()
  • get()
  • copy()
  • to_str()

program symphony_string_methods;
    int x;
    str name, name2;
    x = 100;
    name = "Ali";
    copy(name, name2);
    println(length(name));
    println(get(name, 1));
    println(name2);
    println(to_str(x));
